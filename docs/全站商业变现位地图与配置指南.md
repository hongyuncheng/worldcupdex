# 全站商业变现位地图与配置指南

本文档记录了目前 WorldCupDex 网站上**所有的**商业变现入口。在未来的运营过程中，你可以直接对照此文档进行相应的联盟链接替换或下线操作。

---

## 1. 亚马逊联盟 (Amazon Associates) 推广位
目前网站的商品变现主要依赖亚马逊联盟，采用“搜索页策略”防断货。数据统一配置在 `data/affiliate-products.json`，跳转逻辑由 `/api/track-affiliate.get.ts` 统一接管和记录。

### 1.1 球队专属页面 - 底部商品推荐区
*   **位置**：`pages/teams/[id].vue` -> 页面底部的 `<JerseyRecommend />` 组件。
*   **触发条件**：进入阿根廷、巴西等球队主页。
*   **展示内容**：展示对应球队的 Amazon 搜索结果页（如“Explore Argentina Fan Shop”）。如果当前球队没有配置链接，会自动显示 `generic`（通用）的兜底链接。
*   **修改方式**：直接修改 `data/affiliate-products.json` 中对应 `teamId` 的数据。

### 1.2 球队专属页面 - 顶部快捷按钮
*   **位置**：`pages/teams/[id].vue` -> 球队名片下方，黄色的“Become a Fan”按钮旁边的紫色“👕 Buy [Team] Official Jersey”按钮。
*   **触发条件**：用户一进入球队主页即可见。
*   **跳转逻辑**：点击后会通过 `/api/track-affiliate?teamId=xxx&partner=Amazon` 中转，最终跳转到和“底部商品推荐区”一模一样的亚马逊链接。
*   **修改方式**：自动读取 `data/affiliate-products.json`，无需单独修改。

### 1.3 球队专属页面 - 球员卡片弹窗
*   **位置**：`pages/teams/[id].vue` -> 点击某个球员弹出的 Modal 弹窗底部的“👕 Buy Jersey”按钮。
*   **跳转逻辑**：与 1.2 相同，跳转到该球队配置的亚马逊链接。

### 1.4 球迷卡与预测结果页
*   **球迷身份卡结果页**：`pages/fan-card/result.vue` 底部。
*   **单场比赛预测结果页**：`pages/predict/[id].vue` 底部（用户生成卡片后展示）。
*   **触发逻辑**：根据当前选择的球队（如阿根廷），自动读取对应的亚马逊链接。
*   *(注：IQ挑战结果页 `pages/quiz/result.vue` 目前由于没有特定的关联球队，暂时未放置商品推广位。)*

### 1.5 全局/赛程页面 - 通用看球神器 (原球票按钮的替代品)
*   **位置 A**：`components/MatchCard.vue` -> 每个对阵卡片底部的“🎟️ Find Tickets”按钮。
    *   **历史背景**：原来是指向 StubHub 的门票搜索。
    *   **现状**：现在点击该按钮，会跳转到亚马逊的“Watch Party Essentials”（比如投影仪、啤酒机），因为 Skimlinks 暂未通过。
*   **位置 B (待放置)**：`components/WatchPartyGear.vue`。
    *   这是一个我们刚刚写好的独立组件，里面陈列了高单价的投影仪、啤酒机。你可以把它放置在首页或赛程页的底部。
*   **修改方式**：在 `data/affiliate-products.json` 中修改 `teamId: "global"` 对应的数据。

## 2. 谷歌广告 (Google AdSense) 推广位
*由于目前 AdSense 账号还在审核（准备中），所以以下组件均处于**隐藏/未挂载**状态。一旦审核通过拿到 Slot ID，即可一键开启。*

### 2.1 广告位组件
*   **文件**：`components/AdSlot.vue`
*   **说明**：封装了 `<ins class="adsbygoogle">` 的标准组件。

### 2.2 预留的展示位置 (待定)
一旦 AdSense 审核通过，我们计划在以下几个高流量位置插入 `<AdSlot />`：
1.  **首页/赛程页信息流中**：在比赛卡片列表的第 3 个和第 4 个卡片之间。
2.  **预测结果生成页 (Result Page)**：在用户生成卡片、准备下载图片之前的高视觉停留区。
3.  **博客文章详情页**：文章内容的中间或侧边栏（如果屏幕足够宽）。

## 💡 日常维护指南
*   **换链接**：只需要动 `data/affiliate-products.json`，不要动代码。
*   **加新球队链接**：在 JSON 文件里照猫画虎复制一段，把 `teamId` 改成国家名拼音全小写（如 `japan`），换上亚马逊链接即可。
