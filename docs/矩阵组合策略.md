## 四、WorldCupDex 功能升级方案

### 做 KickIQ 没有的功能，形成差异化

#### 4.1 每日世界杯 IQ 测试（推荐优先开发）

**概述**：每天 5 道知识问答题，从现有数据库自动生成。

**题目来源**（全部基于现有数据，零成本）：
- "哪支球队的 FIFA 排名是第 15？"（teams.json → fifaRank）
- "阿根廷的主教练是谁？"（teams/argentina.json → coach）
- "日本在哪个小组？"（teams.json → group）
- "以下哪位是巴西队球员？"（teams/brazil.json → squad）
- "2026世界杯有多少支球队参赛？"（常识题）

**用户流程**：
```
进入 /quiz → 5题限时答题（每题15秒）→ 得分（0-100）→ IQ等级卡片 → 分享
```

**IQ 等级**：
| 分数 | 等级 | 标签 |
|------|------|------|
| 0-20 | 伪球迷 | "世界杯？什么杯？" |
| 21-40 | 路人球迷 | "至少知道梅西" |
| 41-60 | 休闲球迷 | "四年看一次" |
| 61-80 | 资深球迷 | "数据都在脑子里" |
| 81-100 | 足球教授 | "你比解说员还懂" |

**病毒点**：
- 每日不同题目，天天可以刷
- 分享 IQ 分数卡 → "你的世界杯 IQ 是多少？"
- 好友对比："我 85 分，你敢挑战吗？"

**与 KickIQ 互导**：
- IQ 测试完成后："想知道你像哪支球队？→ KickIQ Quiz"
- KickIQ Quiz 完成后："测测你对[球队]了解多少？→ WorldCupDex IQ"

#### 4.2 简单比分竞猜

**概述**：每场比赛开赛前，用户预测比分。赛后自动判分，积分累计排名。

**核心规则**：
- 猜对胜/平/负：+3 分
- 猜对精确比分：+10 分
- 每日可预测当天所有比赛（无次数限制）
- 积分累计 → 全站排行榜

**积分用途（闭环）**：
- 全球排名展示
- 称号系统：预测新手 → 预测达人 → 预测大师 → 预测之神
- 分享战绩卡片

**与 KickIQ 互导**：
- 用户猜完比分后："看看 AI 怎么预测？→ KickIQ Predict"
- KickIQ Predict 完成后："来证明你比 AI 更准 → WorldCupDex 竞猜"

#### 4.3 赛程提醒 + 邮件收集

**概述**：用户订阅关注球队的比赛提醒，通过邮件推送。

**邮件内容**：
- 赛前 1 小时："你关注的 [球队] 即将开赛！点击预测比分 →"
- 赛后："比赛结果：[比分]。看你的预测是否正确 →"
- 每周："本周 IQ 挑战排名：你是第 X 名"

**价值**：
- 邮件列表是赛后最有价值的长期资产
- 可与 KickIQ 共享列表（用户同意前提下）
- 每封邮件内可放广告位和联盟链接

## 九、风险与应对

| 风险 | 概率 | 应对 |
